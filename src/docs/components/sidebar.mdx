# Sidebar

A responsive navigation sidebar component with collapsible states and nested items.

## Usage

```tsx
import { Sidebar } from "@/components/Sidebar/Sidebar";
import { Home, Settings, Users } from "lucide-react";

export default function SidebarDemo() {
  return (
    <Sidebar
      items={[
        {
          icon: <Home className="h-4 w-4" />,
          label: "Home",
          href: "/",
        },
        {
          icon: <Users className="h-4 w-4" />,
          label: "Users",
          items: [
            {
              label: "Overview",
              href: "/users",
            },
            {
              label: "Settings",
              href: "/users/settings",
            },
          ],
        },
      ]}
    />
  );
}
```

## Props

- `items` - Array of navigation items
  - `icon` - Optional icon element
  - `label` - Text or element for the item
  - `href` - Optional URL for navigation
  - `items` - Optional nested navigation items
  - `disabled` - Whether the item is disabled
- `collapsed` - Control collapsed state
- `onCollapse` - Callback when collapse state changes
- `defaultCollapsed` - Initial collapsed state
- `className` - Additional CSS classes

### Controlled Collapse

```tsx
function ControlledSidebar() {
  const [collapsed, setCollapsed] = React.useState(false);

  return (
    <Sidebar
      collapsed={collapsed}
      onCollapse={setCollapsed}
      items={[
        {
          icon: <HomeIcon />,
          label: "Home",
          href: "/",
        },
      ]}
    />
  );
}
```

## Best Practices

1. Navigation Structure

   - Keep hierarchy clear and logical
   - Limit nesting depth
   - Group related items together
   - Use consistent patterns

2. Visual Design

   - Clear visual hierarchy
   - Consistent spacing
   - Obvious active states
   - Clear hover states

3. Interaction

   - Smooth transitions
   - Clear feedback
   - Easy collapse/expand
   - Keyboard support

4. Responsive Design

   - Consider mobile viewports
   - Handle overflow properly
   - Maintain usability when collapsed
   - Consider touch targets

5. Performance
   - Optimize icons
   - Minimize re-renders
   - Handle large item lists
   - Efficient animations
