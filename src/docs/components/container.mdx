# Container

A responsive container component that centers content and provides consistent max-widths at different breakpoints.

## Usage

```tsx
import {
  Container,
  PageContainer,
  SectionContainer,
  ContentContainer,
  NarrowContainer,
  FullWidthContainer,
} from "@/components/Container/Container";

export default function ContainerDemo() {
  return (
    <Container>Content will be centered with responsive max-width</Container>
  );
}
```

## Props

- `size` - Maximum width at breakpoints
  - `sm` - 640px
  - `md` - 768px
  - `lg` - 1024px (default)
  - `xl` - 1280px
  - `2xl` - 1536px
  - `full` - 100%
- `padding` - Apply horizontal padding (default: true)
- `centered` - Center the container (default: true)
- `as` - HTML element to render as ('div' | 'section' | 'article' | 'main')
- `className` - Additional CSS classes

## Semantic Containers

### PageContainer

Main container for page layouts. Uses xl (1280px) max-width.

```tsx
function PageLayout() {
  return (
    <PageContainer>
      <h1>Page Title</h1>
      <p>Page content goes here...</p>
    </PageContainer>
  );
}
```

### SectionContainer

Container for major page sections. Uses lg (1024px) max-width.

```tsx
function PageWithSections() {
  return (
    <PageContainer>
      <SectionContainer className="py-12">
        <h2>Hero Section</h2>
        {/* Hero content */}
      </SectionContainer>

      <SectionContainer className="py-12 bg-muted">
        <h2>Features</h2>
        {/* Features grid */}
      </SectionContainer>
    </PageContainer>
  );
}
```

### ContentContainer

Container for content areas. Uses md (768px) max-width.

```tsx
function BlogPost() {
  return (
    <PageContainer>
      <ContentContainer className="py-8">
        <h1>Blog Post Title</h1>
        <div className="prose">{/* Blog content */}</div>
      </ContentContainer>
    </PageContainer>
  );
}
```

### NarrowContainer

Container for narrow content. Uses sm (640px) max-width.

```tsx
function LoginForm() {
  return (
    <PageContainer>
      <NarrowContainer className="py-8">
        <h2>Login</h2>
        <form>{/* Form fields */}</form>
      </NarrowContainer>
    </PageContainer>
  );
}
```

### FullWidthContainer

Container that spans the full width while maintaining padding.

```tsx
function FullWidthSection() {
  return (
    <PageContainer>
      <FullWidthContainer className="bg-primary text-white py-12">
        <SectionContainer>
          <h2>Full Width Banner</h2>
          {/* Banner content */}
        </SectionContainer>
      </FullWidthContainer>
    </PageContainer>
  );
}
```

## Best Practices

1. Container Selection

   - Use PageContainer for main page wrapper
   - Use SectionContainer for major page sections
   - Use ContentContainer for article/text content
   - Use NarrowContainer for forms and focused content
   - Use FullWidthContainer for edge-to-edge sections

2. Layout Structure

   - Nest containers appropriately
   - Maintain consistent spacing
   - Use semantic HTML elements
   - Consider content width for readability

3. Responsive Design

   - Test at all breakpoints
   - Use appropriate padding
   - Consider mobile-first design
   - Handle overflow properly

4. Performance

   - Avoid deeply nested containers
   - Use appropriate semantic containers
   - Consider content reflow
   - Optimize for layout shifts

5. Accessibility
   - Use semantic HTML
   - Maintain proper heading structure
   - Consider reading width
   - Ensure proper spacing
